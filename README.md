# 🛡️ NetGer App v1.0

**Firewall & Network Administration Tools**

---

## 📖 Deskripsi

**NetGer App** adalah aplikasi tools administrasi jaringan yang membantu Network Administrator dalam:

- 🔥 **Generate konfigurasi Firewall** (FortiGate & CheckPoint)
- 🌐 **NSLookup massal** dengan multi-threading
- 📊 **Export hasil ke Excel**
- ⚡ **Multi DNS Server support**

---

## ✨ Fitur Utama

### 🔥 Firewall Command Generator
- Generate script konfigurasi FortiGate Address
- Generate script konfigurasi FortiGate Group
- Generate script konfigurasi CheckPoint
- Support multiple IP format
- One-click copy to clipboard

### 🌐 NSLookup Pro Tool
- Multi DNS Server (Google, Cloudflare, Quad9, OpenDNS)
- Multi-threading (1-20 threads)
- Configurable timeout
- Real-time logging
- Progress bar
- Export hasil ke Excel (.xlsx)
- Support multiple IP detection

### 🎨 UI/UX
- Modern dark theme
- Responsive design (desktop & mobile)
- Smooth animations
- Keyboard shortcuts (Ctrl+1, Ctrl+2)
- Toast notifications

## 🚀 Cara Install

### Metode 2: Portable
1. Download `NetworkTools.exe`
2. Simpan di folder manapun
3. Double-click untuk menjalankan
4. Buka browser → `http://127.0.0.1:5000`

### Metode 3: Dari Source Code
```bash
# Clone repository
git clone https://github.com/joyravelo/netger-app.git

# Install dependencies
pip install -r requirements.txt

# Jalankan aplikasi
python network-tools.py

# Buka browser
http://127.0.0.1:5000
